# Handwritten digit recognition

Web application that recognizes handwritten digits. 

## Technologies

* Python 3.x
* Flask
* Scikit-learn

## Usage

To run up you need to have `python` installed and all modules from `requirements.txt`

Then run:

```
python app.py
```

Server will be running on 

```
http://127.0.0.1:5000
```
